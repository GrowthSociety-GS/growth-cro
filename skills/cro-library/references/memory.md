## Error & Failure Logs

### Learning from what doesn't work

```
FAILURES & LESSONS LEARNED

[Pattern/Template]: [Name] — Created [date]

Failure Mode 1: Low conversion in e-commerce context
├─ Expected: Template designed for e-commerce jewelry
├─ Actual: 2.1% conversion (target 5.5%)
├─ Root cause: Hero design too minimalist for luxury segment
│   └─ Luxury buyers need emotional resonance, not just clarity
├─ Learning: Minimalism ≠ universally applicable (sector-dependent)
├─ Fix applied: Create luxury-variant with more visual richness
└─ Status: RESOLVED (variant created and tested successfully)

Failure Mode 2: Form friction too high
├─ Expected: 5-field qualification form (enterprise model)
├─ Actual: 12% abandonment rate at form field
├─ Root cause: Cold traffic doesn't want to qualify before understanding
├─ Learning: Form field ordering matters (save qualifying questions for later)
├─ Fix applied: Reduce to 2 fields upfront, 3 fields after CTA click
└─ Status: RESOLVED (drop abandoned rate to 4%)

[Common Failure Patterns Across Library]:

Pattern: "Minimalism works everywhere"
├─ Reality: Luxury, emotional categories need richness
├─ Reality: Unaware audiences need more copy/explanation
├─ Fix: Always contextualize to sector + audience awareness

Pattern: "Long-form copy is always high-friction"
├─ Reality: Long-form works if writing is excellent and relevant
├─ Reality: Short copy fails if it's vague or doesn't resonate
├─ Fix: Quality > length. Bad short copy < good long copy

Pattern: "Video always improves conversion"
├─ Reality: Bad video performs worse than no video
├─ Reality: Product demos work, lifestyle videos don't always
├─ Fix: Only use video if it genuinely clarifies/persuades, not for decoration

⚠️ Systemic Issues:
├─ Library over-indexed on SaaS (bias risk)
├─ Very few e-commerce templates (gap)
├─ Minimal lead-gen templates (gap)
└─ Recommendation: Actively recruit templates from underrepresented sectors
```
