# Task CR-05 — Screenshot QA

## Description

Module `moteur_gsg/renderer_qa/` qui capture screenshots desktop + mobile via MCP Playwright, puis appelle Sonnet vision sur les 2 images pour score visual_ambition + detect issues.

Source : PRD §FR-5 + Codex addendum §"Screenshot judge: Mandatory".

## Acceptance Criteria

- [ ] **Prerequisite** : MCP Playwright installed and configured. Check `.claude/settings.json` ou `~/.claude/mcp.json` has playwright MCP server entry. Si absent, AGENT STOPS et requests Mathis install.
- [ ] Module `moteur_gsg/renderer_qa/` créé
- [ ] `moteur_gsg/renderer_qa/screenshot_capture.py` (axe API client, ≤150 LOC) :
  - `capture_dual_viewport(html_path: Path, output_dir: Path) -> ScreenshotSet`
  - Desktop : 1440x900 (default)
  - Mobile : 375x667 (iPhone SE-ish)
  - Full page screenshot (pas just viewport)
  - Returns `ScreenshotSet` Pydantic (desktop_path, mobile_path, captured_at)
- [ ] `moteur_gsg/renderer_qa/screenshot_inspector.py` (axe API client, ≤200 LOC) :
  - `inspect(screenshots: ScreenshotSet, route_contract: CreativeRouteContract) -> ScreenshotQAReport`
  - Sonnet vision call avec les 2 PNGs encoded base64
  - System prompt : "tu es un senior art director, score cette LP sur 5 axes (visual_ambition, hierarchy, mobile_fit, text_overflow_risk, blank_areas), liste problèmes critiques"
  - Output JSON parsé en `ScreenshotQAReport` Pydantic
- [ ] Modèle `ScreenshotQAReport` Pydantic frozen :
  - `screenshots: ScreenshotSet`
  - `visual_ambition_score: int (1-10)`
  - `hierarchy_score: int (1-10)`
  - `mobile_fit_score: int (1-10)`
  - `text_overflow_risk: Literal["none", "low", "medium", "high"]`
  - `blank_areas_detected: list[str]` (descriptions)
  - `critical_issues: list[str]`
  - `gate_passed: bool` (= visual_ambition_score ≥6 AND no critical_issues)
  - `inspector_rationale: str (min_length=100)`
- [ ] Persist `data/captures/<client>/<page_type>/screenshot_qa_report.json` + PNGs dans `screenshots/` subdir
- [ ] Tests pytest :
  - `test_screenshot_capture.py` : fixture HTML simple → 2 PNGs créés, dimensions correctes (MOCKED MCP — pas de real Playwright en CI)
  - `test_screenshot_inspector.py` : fixture ScreenshotSet + mocked Sonnet vision → ScreenshotQAReport valid

- `moteur_gsg/renderer_qa/screenshot_capture.py` (~140 LOC)

**MCP Playwright** :
- Server-level config (hors compte des 8 skills/session per CLAUDE.md anti-pattern #12)
- Install : check official MCP Playwright docs ou utiliser `npx @playwright/mcp` ou similaire
- Wire in module : import MCP client, call `playwright.screenshot(url, viewport)` async

**Coût Sonnet vision** : ~$0.01 per inspection (2 PNG images encoded + ~2K text prompt + ~1K output). Negligeable.
