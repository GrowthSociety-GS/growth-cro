# Task CR-03 — Contracts schemas

## Description

Schemas Pydantic v2 stricts qui forment l'interface frontière inter-modules (Composer → Renderer → QA). Cf. CODE_DOCTRINE §TYPING.

Source : PRD §FR-3.

## Acceptance Criteria

- [ ] Extend `growthcro/models/creative_models.py` (CR-01 created, CR-03 étend) ou split si dépasse 200 LOC en :
  - `growthcro/models/creative_route_models.py` (CR-01 + CR-02)
  - `growthcro/models/visual_composer_models.py` (CR-03)
- [ ] `CreativeRouteContract` Pydantic v2 frozen+extra=forbid :
  - `route_id`, `route_name`
  - `all thesis fields from CreativeRoute (CR-01)` (frozen at this point, deterministic)
  - `decision_metadata` (selected_at, model_used, scores)
- [ ] `VisualComposerContract` Pydantic v2 frozen :
  - `route_id` (foreign key to CreativeRouteContract)
  - `sections: list[SectionSpec]` — each : `section_id`, `section_type` (hero/proof_strip/listicle_item/testimonials/comparison/faq/final_cta/etc), `layout`, `copy_slots`, `visual_modules`
  - `modules: list[VisualModule]` — each : `module_id`, `module_type` (gradient_bg/grain_overlay/dashboard_panel/etc), `target_section`, `params`
  - `motion_specs: list[MotionSpec]` — each : `target_selector`, `trigger` (scroll/hover/load), `animation_type` (fade/slide/scale/etc), `duration_ms`, `easing`, `delay_ms`
  - `asset_specs: list[AssetSpec]` — each : `asset_id`, `asset_type` (hero_image/texture/illustration/logo/background), `prompt_for_image_gen`, `target_dimensions`, `placement`
  - `responsive_breakpoints` (mobile, tablet, desktop)
- [ ] Validators : section_ids uniques, module target_section pointe vers section_id valide, motion target_selector parsable
- [ ] Re-exports dans `moteur_gsg/creative_engine/__init__.py` (CR-01 init)
- [ ] Tests pytest :
  - `test_contracts.py` : schemas frozen rejette mutation, extra fields rejected, invariants (unique IDs, foreign keys) validés
