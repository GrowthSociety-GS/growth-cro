# Task: Create skill `growthcro-prd-planner`

## Description

Créer un **vrai skill Anthropic** (invocable via Skill tool) qui transforme une idée / module / sprint en artefacts CCPM structurés : **PRD + Epic + Issues atomiques**, avec templates et acceptance/dependencies/stop-conditions systématiques.

Le skill est CCPM-aware (conforme `~/.claude/skills/ccpm/`), mais ajoute la doctrine GrowthCRO en plus : 8 axes mono-concern, LOC limits, anti-patterns CLAUDE.md, format anti-drift de `growthcro-anti-drift`.

Source : `AUDIT_DECISION_DOSSIER_2026-05-16` §5.4 + `SKILLS_AND_AGENT_STACK_ADDENDUM.md` §4.2.

  - Anti-patterns CLAUDE.md à vérifier
  - Format anti-drift de `growthcro-anti-drift` injecté dans chaque issue
  - Référence à `AUDIT_DECISION_DOSSIER_*` comme source de vérité contextuelle
