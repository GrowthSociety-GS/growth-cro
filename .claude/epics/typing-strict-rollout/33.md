### Config mypy

Vérifier d'abord si `pyproject.toml` existe au repo root :

```bash

```

Si oui, éditer. Si non, créer un minimal :

```toml
[tool.mypy]
python_version = "3.11"
ignore_missing_imports = true
warn_unused_configs = true
strict = false  # global reste loose pour absorption progressive

[[tool.mypy.overrides]]
module = [
  "moteur_gsg.core.visual_intelligence",
  "moteur_gsg.core.context_pack",
  "growthcro.recos.orchestrator",
  "growthcro.models.*",
]
strict = true
disallow_any_generics = true
disallow_untyped_defs = true
warn_return_any = true
```

### scripts/typecheck.sh

```bash
#!/usr/bin/env bash
# typecheck.sh — guard mypy --strict sur scope défini
set -euo pipefail

STRICT_TARGETS=(
  "moteur_gsg/core/visual_intelligence.py"
  "moteur_gsg/core/context_pack.py"
  "growthcro/recos/orchestrator.py"
  "growthcro/models/"
)

GLOBAL_BUDGET=55  # absorption progressive 88 → ≤55

echo "→ mypy --strict on top-3 + models..."
mypy --strict "${STRICT_TARGETS[@]}" || { echo "❌ strict scope failed"; exit 1; }

echo "→ mypy global budget check..."
total=$(mypy growthcro/ moteur_gsg/ moteur_multi_judge/ skills/ 2>&1 | grep -cE "^.*: error:" || echo 0)
if [ "$total" -gt "$GLOBAL_BUDGET" ]; then
  echo "❌ mypy errors $total > budget $GLOBAL_BUDGET"
  exit 1
fi
echo "✓ mypy: $total errors (budget $GLOBAL_BUDGET)"
```
