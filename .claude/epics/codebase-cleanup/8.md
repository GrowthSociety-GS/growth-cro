# Task: Split GSG (moteur_gsg) god files

## Description

The Growth Site Generator engine (`moteur_gsg/`) carries the heaviest god files in the codebase: `controlled_renderer.py` (1,665 LOC) and `mode_1_persona_narrator.py` (1,448 LOC). Plus the GSG-side scripts: `gsg_generate_lp.py` (1,218 LOC) and `enrich_v143_public.py` (1,339 LOC) — the latter may already have been handled by Task 005; if so, only the moteur_gsg files are in scope here.

GSG is in crisis per V26.AF (one of the doctrine immutables: hard limit prompt persona_narrator ≤8K chars). Splits must respect this — the prompt-assembly concern in `mode_1_persona_narrator.py` MUST surface a single function whose output is asserted to be ≤8K chars, with a unit-style assertion at module load (not a unit test — out of scope — but a runtime guard).

- `moteur_gsg/modes/mode_1_persona_narrator.py` (1,448 LOC)
