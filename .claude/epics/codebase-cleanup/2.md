# Task: Inventory & parity instrumentation

## Description

Wave 1 prelude. Before any code moves, lock in the regression contract: a bit-level parity check on one canonical client (`weglot`) covering the full pipeline (capture → perception → score → recos). Verify the existing instruments (`audit_capabilities.py`, `state.py`, `SCHEMA/validate_all.py`) are green and behave as the rest of the epic assumes. Create the dedicated branch + worktree.

## Acceptance Criteria

- [ ] `epic/codebase-cleanup` branch created from up-to-date `main`; worktree at `../epic-codebase-cleanup/`.
- [ ] `scripts/parity_check.sh <client>` exists, runs end-to-end on `weglot`, snapshots:
  - perception JSON for each captured page
  - score JSON (pillars + page-type) for each captured page
  - reco JSON for each captured page
- [ ] Snapshots stored under `_archive/parity_baselines/<client>/<ISO-date>/`.
- [ ] Re-running the script on the same code/data exits 0 (deterministic on the meaningful fields — timestamps masked).
- [ ] `python3 scripts/audit_capabilities.py` runs, output saved to `_archive/parity_baselines/audit_capabilities_baseline.json`.
- [ ] `python3 state.py` runs; output saved to `_archive/parity_baselines/state_baseline.txt`.
- [ ] `python3 SCHEMA/validate_all.py` exits 0; result logged.
- [ ] A 1-page `docs/state/CLEANUP_BASELINE_2026-05-08.md` documents what was snapshotted, how to compare, and the canonical client choice.

## Technical Details

- Parity script is plain bash + `jq`/`diff`. No new Python dependencies.
- Mask volatile fields: `timestamp`, `generated_at`, `mtime`, `run_id`, any UUIDs. Keep score values, reco IDs, payload structure.
- The snapshot directory becomes the source of truth — every subsequent task re-runs the parity script and diffs against this baseline.
- If `audit_capabilities.py` heuristics look broken (e.g. miscount), file an inline note in the baseline doc but **do not fix in this task** — fix lands in Task 9 where orphans are addressed.
- Branch hygiene: no other tasks may commit until this one merges its baseline.
