> Before coding, invoke `growthcro-anti-drift`. Source: `.claude/skills/growthcro-anti-drift/SKILL.md`.

## Description

Rebuild `webapp/apps/shell/app/page.tsx` (la home) en **Command Center actionnable 4 zones** au lieu du mur dense de KPIs/panels actuel. Réponse à la question "qu'est-ce que je dois faire maintenant ?" en <30s.

Source : [`WEBAPP_TARGET_IA_2026-05.md`](../../docs/state/WEBAPP_TARGET_IA_2026-05.md) §1.1 + PRD §FR-8 + User Story 1 (Mathis 9h café, pilote sa journée).

## Acceptance Criteria

- [ ] **Drop le mur dense actuel** : DashboardTabs (KpiSummary + ChartsRow + PrioritiesPanel + etc.), legacy ClientHeroDetail, FleetPanel dense → archived ou réduits drastiquement
- [ ] **4 zones canoniques** dans cet ordre :
  1. **Today / Urgent actions** (top) — 3-5 actions ranked : clients à risque (audit score <60), runs failed récents, recos validées à shipper, captures stalled
  2. **Fleet Health** — KPIs sobres : X clients audités, score moyen, runs réussis dernière 24h, GSG runs récents
  3. **Recent Runs** — list 5-10 derniers runs avec statut realtime (Realtime channel public:runs préservé)
  4. **Next Best Actions** — suggestions produit : "Auditer Y nouveau client", "Validate Z recos pending", "Ship GSG draft W" (logic simple : prioritisation backlog)
- [ ] **Empty states honnêtes** via `EmptyState` (B2) — si aucune action urgente, "Aucune action urgente — your fleet looks healthy 🟢"
- [ ] **Drill-down** : chaque item de chaque zone wire vers la route appropriée
- [ ] **Performance** : query budget — pas plus de 5 queries Supabase parallèles (Phase I1 will measure baseline)
- [ ] **Worker badge global** : déjà placé en B1 slot `gc-worker-health-slot`, pas besoin de toucher (B3 wire la logic)
- [ ] **Module Maturity** : zones qui dépendent de modules `ready_to_configure` ou `no_data` affichent honnêtement (e.g. "Connect Reality to see fleet health KPIs" CTA)
- [ ] `npm run typecheck` exit 0
- [ ] `npm run lint` exit 0
- [ ] `python3 scripts/lint_code_hygiene.py --staged` exit 0
- [ ] Commit conventional : `feat(webapp): rebuild home as Command Center 4 zones [#74]`
