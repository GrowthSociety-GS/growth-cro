# Task: Skill Integration Blueprint (16 skills, combo packs anti-cacophonie)

## Description

Définir le plan d'intégration concret des 16 skills audités (8 essentiels / 6 on-demand / 5 exclus). Pour chaque essentiel : où il se branche (audit/GSG/webapp), comment (auto-load ou invocation), avec quels autres skills il forme un combo cohérent. Anti-cacophonie : règle "1 parti pris visuel par projet" codifiée comme anti-pattern #12 dans CLAUDE.md.

Référence PRD : FR-2.
