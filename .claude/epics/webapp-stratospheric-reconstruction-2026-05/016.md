### Architecture-explorer-data update strategy
The file is 5683 lines JS data. Use careful Edit to replace specific sections without breaking JSON structure :
```python
# scripts/update_architecture_explorer.py (NEW small utility)
import json, re
raw = open('deliverables/architecture-explorer-data.js').read()
marker = raw.find('= {')
data = json.loads(raw[marker+2:].rstrip().rstrip(';').strip())

# Remove 5 µfrontend module entries
data['modules'] = [m for m in data['modules'] if m['package'] != 'webapp/microfrontend']

# Add single shell entry
data['modules'].append({
    'id': '@growthcro/shell',
    'package': 'webapp/apps/shell',
    'path': 'webapp/apps/shell/',
    'status': 'active',
    'lifecycle_phase': 'runtime',
    'purpose': 'Consolidated Next.js 14 App Router shell (post-D1.A monorepo decision 2026-05-14)',
    # ...
})

# Update counts
data['counts']['modules'] = len(data['modules'])
data['meta']['revision_notes'] = data['meta'].get('revision_notes', [])
data['meta']['revision_notes'].append({
    'date': '2026-05-14',
    'note': 'D1.A monorepo confirmed — 5 µfrontends collapsed to 1 shell'
})

# Write back
new_js = f'window.ARCH = {json.dumps(data, indent=2, ensure_ascii=False)};\n'
open('deliverables/architecture-explorer-data.js', 'w').write(new_js)
```
