# Task 011 : reality-layer-5-connectors-wiring

## Description

Wire les 5 connectors Reality Layer live (actuellement stubbed). Source : V26 HTML L2840-2894 (`renderReality`) + architecture canonique. Connecte 5 OAuth flows : Catchr (Meta Ads) · Meta Ads direct · Google Ads · Shopify · Microsoft Clarity. Permet d'afficher les CVR/CPA/AOV réels par client (ground truth pour cycle apprenant V29+V30).

Bloquant Mathis : OAuth creds + app registrations.

## Acceptance Criteria

- [ ] **5 OAuth callback handlers** (`webapp/apps/shell/app/api/auth/[connector]/callback/route.ts`) :
  - Per connector : exchange auth code → access_token + refresh_token
  - Store encrypted in Supabase `client_credentials` table (RLS strict per-client per-user)
- [ ] **Supabase migration** `supabase/migrations/20260513_0016_reality_layer.sql` :
  - `client_credentials` table : id, client_id, connector text check (5 values), access_token_encrypted, refresh_token_encrypted, expires_at, scope, created_by, updated_at
  - `reality_snapshots` table : id, client_id, connector, metric text (cvr/cpa/aov/traffic/impressions), value numeric, ts, raw_response_json
  - Encryption via Supabase Vault or pgcrypto
- [ ] **Credentials gate UI** (`<CredentialsGateGrid />` NEW) per V26 :
  - 5 connector cards : status pill (connected/expired/not_connected) + "Connect" CTA → OAuth flow
  - Per card : last sync timestamp + scopes granted
- [ ] **Reality polling worker** : extend `growthcro/worker/dispatcher.py` (task 002) :
  - run.type='reality' → for each connected connector × client → fetch latest metrics → insert reality_snapshots
  - Cron schedule : every 1h (Vercel cron job, configured in `vercel.json`)
- [ ] **`/reality/[clientSlug]` route** (`webapp/apps/shell/app/reality/[clientSlug]/page.tsx` enhance existing) :
  - Per metric chart (sparkline 30j) : CVR · CPA · AOV · Traffic · Impressions
  - Per connector breakdown
  - Last update timestamp
- [ ] **`/reality` fleet view** (`webapp/apps/shell/app/reality/page.tsx` enhance) :
  - 51 clients × 5 metrics × 5 connectors heat map
  - Click cell → drilldown
